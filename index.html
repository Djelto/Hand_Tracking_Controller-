Installation en 4 étapes

1
Installer Python 3.12
Téléchargez Python 3.12 sur python.org et installez-le. Cochez bien "Add Python to PATH" pendant l'installation.
2
Installer les dépendances
Ouvrez un terminal et exécutez :
Terminal Windows
py -3.12 -m pip install mediapipe==0.10.14 opencv-python pyautogui numpy ipykernel
3
Créer le kernel Jupyter
Enregistrez Python 3.12 comme kernel Jupyter :
Terminal Windows
py -3.12 -m ipykernel install --user --name=Python312 --display-name "Python 3.12 (MediaPipe)"
4
Ouvrir le notebook & changer le kernel
Ouvrez Hand_Tracking_Controller.ipynb dans Jupyter. Allez dans Kernel → Change Kernel → Python 3.12 (MediaPipe). Exécutez les cellules de 1 à 7 dans l'ordre. La fenêtre caméra s'ouvre à la Cellule 7 — montrez votre main !
